# Library Book Issue & Fine Management System

## 📌 Project Overview
The Library Book Issue & Fine Management System is a MySQL-based database project designed to efficiently manage books, members, issue records, and fine calculations in a structured manner.

The system implements relational database design using primary keys and foreign keys to maintain proper relationships between tables. It ensures data integrity, reduces redundancy, and enables accurate tracking of issued books and unpaid fines.

## 🚀 Features
- Book Management (Add, Update, Track Availability)
- Member Management
- Book Issue & Return Tracking
- Automatic Fine Calculation
- View Issued Books
- View Pending/Unpaid Fines

## 🛠 Technologies Used
- MySQL
- SQL
- MySQL Workbench

## 🗄 Database Concepts Implemented
- Relational Database Design
- Primary and Foreign Keys
- Constraints (NOT NULL, AUTO_INCREMENT)
- JOIN Operations
- Subqueries
- Aggregate Functions
- Views
- Transactions
